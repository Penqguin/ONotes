## Reflection
In Java, random numbers are typically generated using the static Math.random() method or the java.util.Random class.
**The Math.random() is a static method that returns a double value greater than or equal to 0.0 and less than 1.0 java.util.Random class is the most common method where you create an instance of the class and use its methods to generate various data types (int, double, long, boolean, etc.)**
