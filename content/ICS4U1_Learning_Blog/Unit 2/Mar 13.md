# Control Structures (Loops)

## Reflection
Iteration is the power of computing, allowing for repetitive tasks to be handled with minimal code.

### Key Concepts and Notes
- `for`, `while`, `do-while`
- `break`, `continue`

## Examples
```java
for(int i=0; i<10; i++) { ... }
```
